  • Flexbox
    1. One dimensional
    2. Allows to distribute space dynamicaly across emlements of an unknown size, hence the term “flex”
    3. Flex items can stretch to the height of the flex container, meaning that equal height columns were possible.
  • Grid
    1. Multi dimensional
    2. Grid property offers a grid-based layout system having rows and columns. It makes the designing of web pages easy without positioning and floating
    3. We can use grid to create a similar effect to flexbox, while still keeping the content arranged in strict rows and columns.
