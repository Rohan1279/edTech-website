  • Flexbox
    1. One dimensional
    2. Allows to distribute space dynamicaly across emlements of an unknown size, hence the term “flex”
    3. Flex items can stretch to the height of the flex container, meaning that equal height columns were possible.
  • Grid
    1. Multi dimensional
    2. Grid property offers a grid-based layout system having rows and columns. It makes the designing of web pages easy without positioning and floating
    3. We can use grid to create a similar effect to flexbox, while still keeping the content arranged in strict rows and columns.

  • Bootstrap
    1. Bootstrap is the most popular HTML, CSS, and JavaScript framework for building responsive, mobile-first projects on the web.
    2. Bootstrap is known for its responsiveness
    3. Frameworks like Bootstrap have abstracted the creation of components to the point where it compels developers to use only the available patterns provided.
  • Tailwind
    1. Tailwind CSS is a utility-first CSS framework for rapidly building custom user interfaces.
    2. Tailwind CSS typically value the framework’s customizability.
    3. Tailwind CSS: its ability to easily build complex user interfaces without encouraging any two sites to look the same.
